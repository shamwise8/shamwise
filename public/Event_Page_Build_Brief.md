# Event Page Build Brief — Team1 Pudgy Padel Night

**For:** Claude Code build session on shamwise.com
**Purpose:** Standalone event page with sponsor outreach focus
**Target URL:** `shamwise.com/team1-padel-night` (or your preferred slug)
**Deployment:** Vercel auto-deploy from git

---

## Build Goal

A **single-page event site** that serves three audiences in priority order:
1. **Potential sponsors** — primary CTA target. Clear tier breakdown, easy contact.
2. **Curious attendees / RSVP applicants** — credibility + venue feel + format clarity.
3. **Public/social shares** — looks great on X cards, mobile-first.

**Tone:** Premium curated creator gathering. Not bro-culture crypto event. Athletic + lifestyle + builder. Think wellness aesthetic meets sports premium, not web3 conference.

---

## Visual Direction

**Fresh theme — distinct from typical web3 event pages.**

### Color Palette
- **Primary:** Terracotta red `#9B3D2F` (matching Sterling's actual padel court turf — distinctive anchor color)
- **Accent:** Sterling brand orange `#D87B3A` (for buttons, links, highlights)
- **Neutral:** Off-white `#F5EDDF` for backgrounds, soft contrast
- **Dark accent:** Deep charcoal `#1A1A1A` (for headers, text)
- **Optional:** Light blue accent `#A8C4D6` if Pudgy branding needs visual nod

### Typography
- **Headlines:** Bold sans-serif with personality. Suggestions: Inter, Söhne, or something with athletic energy. Avoid generic system fonts.
- **Body:** Clean readable sans-serif. Inter or Geist work well.
- **Optional accent:** A serif (Instrument Serif from Sam's existing stack) for pull quotes / date callouts

### Layout
- **Hero:** Full-bleed photo of Sterling courts (image attached). Title + date + co-host logos overlay.
- **Long scroll, single page.** No sub-navigation needed.
- **Mobile-first.** Most sponsor outreach will be DM'd or shared on phones.
- **Generous whitespace.** Avoid the cramped feel of standard event pages.
- **Photo-led.** 3 venue photos integrated through the page, not just clustered at top.

### Key visual moves
- Hero photo darkened with overlay for text legibility (terracotta court, dramatic LED ceiling lighting)
- Section dividers using the terracotta color, not borders or boxes
- Sponsor tier cards in a clean grid, not bulleted lists
- Photos should bleed edge-to-edge or near it — premium feel, not boxed

## What to AVOID

- **No web3 jargon dump.** Don't list "AVAX, Pudgy, Avalanche, GameFi, etc." in lead copy. The room is curated; the page should signal premium not noisy.
- **No internal budget figures.** Sponsors don't see Team1's $1,600 budget breakdown. The sponsor tiers ($400/$600) are the only money mentioned.
- **No "we're 5th pick" type humility.** Confident voice, not apologetic.
- **No bro-culture imagery.** No clinking glasses stock photos, no nightclub aesthetic. Premium athletic lifestyle.
- **No generic "join the future of web3" CTAs.** Every CTA should be specific (Become a sponsor, Get in touch, See past events).

## Technical Notes for Claude Code

- **Stack:** Next.js + Vercel (assumed — adjust based on shamwise.com actual setup)
- **Routing:** New page at `/team1-padel-night` or similar slug
- **Images:** Optimize using next/image. Hero photo should have priority loading, carousel images lazy-loaded.
- **Venue carousel:** Use a lightweight library (Embla Carousel, Swiper, or hand-roll with React state + setInterval). 3-second auto-rotate, fade or slide transition, pause on hover (desktop) / tap (mobile). Keep accessibility (focus states, manual dot navigation optional).
- **OG meta tags:** Critical for X/social sharing. Use sterling_court.jpeg as og:image.
- **Mobile-first:** Test at 375px width before desktop. Most sponsor outreach happens via DM links opened on phones.
- **No tracking scripts unless already on shamwise.com.** Keep it clean.
- **SEO basics:** Meta title "Team1 Pudgy Padel Night — Bangkok, May 24, 2026", meta description with sponsor angle.
